## Image Naming Convention

For SEO purposes, consider renaming images to include the city name:
- `chain-link-fence-[city]-1.jpeg`
- `vinyl-fence-[city]-1.jpeg`
- etc.

## Recommended Image Sizes

- Hero images: 1920x1080px minimum
- Card images: 800x600px
- Gallery images: 800x600px
- Form images: 600x400px
